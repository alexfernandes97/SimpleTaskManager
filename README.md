# SimpleTaskManager

Este é um pequeno gerenciador de tarefas feito em Java com base no terminal.

## Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Ordenar tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Salvar tarefas em um arquivo

## Como executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/alexfernandes97/SimpleTaskManager.git
   ```
2. Compile o projeto:
   ```bash
   javac *.java
   ```

3. Rode o programa:
   ```bash
   java ToDoList
   ```
