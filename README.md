# SimpleTaskManager

Este é um pequeno gerenciador de tarefas feito em Java com base no terminal.

## Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Marcar tarefas como concluídas
- Remover tarefas

## Como executar

Compile os arquivos `.java`:

```bash
javac ToDoList.java TarefaManager.java Tarefa.java
